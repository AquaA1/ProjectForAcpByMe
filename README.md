## Description

A menu-driven 2D Graphics Editor in C using a 2D character array as the drawing canvas.

## Features

* Add Line
'''text
x1 y1 x2 y2
'''
* Add Rectangle
'''text
x y width height
'''
* Add Circle
'''text
center_x center_y radius
'''
* Add Triangle
'''text
x1 y1 x2 y2 x3 y3
'''
* Modify Existing Objects
* Delete Objects
* List Objects
* Display Canvas

## Canvas Specifications

* Width: 60
* Height: 20
* Maximum Objects: 50


## Menu

```text
1. Add Object
2. Delete Object
3. Modify Object
4. List Objects
5. Display Picture
6. Exit
```
